# Звуковые эффекты

Положите сюда файлы звуков в формате `.ogg` или `.wav`.

## Рекомендуемые имена файлов

| Файл | Когда играет |
|------|--------------|
| `shoot.ogg` | Выстрел игрока |
| `hit.ogg` | Попадание пули |
| `coin.ogg` | Подбор монеты |
| `heal.ogg` | Лечение в магазине |
| `menu.ogg` | Нажатие в меню |

Также поддерживаются русские имена: `выстрел`, `попадание`, `монета`, `лечение`, `меню`.

Если файла нет — игра использует встроенный звук-заглушку.

## Фоновая музыка

Музыку кладите в папку `assets/music/` с именем `music.ogg` или `theme.ogg`.
